# HallGuard

IoT and AI-based hallway railing hazard detection system.

## Structure
- edge_app/ : Raspberry Pi application
- training/ : YOLOv8 training scripts
- dataset/  : raw videos and labeled data
- mobile_app/ : Firebase-based alert app
- results/  : evaluation outputs

## How to run
1. Activate venv "venv\Scripts\activate"
2. python edge_app/main.py
